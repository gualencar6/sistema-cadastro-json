# Sistema de Cadastro com JSON em C#

Este projeto surgiu a partir de estudos em JSON. Foi criado um sistema simples de cadastro feito em C# utilizando JSON para armazenamento de dados.

## Funcionalidades
- Entrada de dados do usuário pelo console
- Serialização (Objeto → JSON)
- Salvamento em arquivo
- Leitura do arquivo JSON
- Desserialização (JSON → Objeto)

## Tecnologias utilizadas
- C#
- .NET
- System.Text.Json

## Como executar
1. Clone o repositório
2. Abra no Visual Studio ou VS Code
3. Execute o projeto
